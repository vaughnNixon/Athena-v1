## Frameworks I Live By
- **Memory-First Architecture:** Separate machine execution logs (AAL/SQLite) from human reporting layers (Markdown).
- **Zero Token Waste:** Load context on-demand; never clutter system prompts with static global files unless necessary.
- **Local Control & Oversight:** Human user retains full authority over remote repository pushes and remote actions.
